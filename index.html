Este não aplica nada no sistema. A ideia é servir como camada visual acima do iptables real.
Importar saída do iptables
Cole a saída do comando no servidor e converta para a visualização acima.
No servidor, execute por exemplo:
sudo iptables -L -n -v --line-numbers
Copie toda a saída do terminal e cole abaixo. Este parser simples lê as chains INPUT/OUTPUT/FORWARD da tabela filter.
Observação: O parser é heurístico e pensado. Para produção, você pode trocar por parsing do iptables-save no backend (PHP/Python) e alimentar esta mesma UI.
Nova regra de firewall Preencha os campos e arraste depois para a cadeia desejada.
Dica: use ACCEPT para portas que realmente precisam ficar abertas e DROP/REJECT para o resto. Depois, arraste as regras para ajustar a ordem de processamento.
